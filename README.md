# generator-tui

## Install yo
`npm install -g yo`

## Install generators
`npm install -g generator-tui`

## Usage

0. create new project
`yo tui:new`

1. create module
`yo tui:module`


2. create page
`yo tui:page`


3. create service
`yo tui:service`


4. create modal
`yo tui:modal`


5. create window
`yo tui:window`
